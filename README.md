简单办法：
1.创建.env文件：
```env
AI_API_KEY=sk-xxxxxxxxxxxxxxxx
AI_BASE_URL=https://yunwu.ai/v1
AI_MODEL=gpt-4o
SECRET_KEY=随机32位字符串（重要！）
```

可选邮件配置：

```env
EMAIL_ENABLED=true
EMAIL_SMTP_HOST=smtp.qq.com
EMAIL_SMTP_PORT=465
EMAIL_SENDER=你的QQ邮箱@qq.com
EMAIL_SENDER_PASS=QQ邮箱授权码
EMAIL_RECEIVER=接收邮箱@example.com
```
2.双击启动.vbs
